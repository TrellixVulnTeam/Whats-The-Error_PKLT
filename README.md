### Setup guidelines :

1. Clone the repository.
   ```git
   git clone https://github.com/thisiskartikgupta/Whats-The-Error.git
   ```
2. Navigate to ./What-The-Error
   ```sh
   cd '.\What-The-Error\'
   ```
3. Copy the file you want to check for errors.
4. Run main.py
   ```sh
   python main.py
   ```
